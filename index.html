Conversão de valores com JavaScript

Hoje, vamos ver oque acontece se executarmos estes comandos no JavaScript {


1) 1 + 1
2) 1 - "1"
3) 1 + "1"


}

Comandos


Primeiro comando

'

Podemos ver que o resultado de 1 + 1 é 2, e isso acontece pois 1 em JavaScript é considerado um número.

Segundo comando

'

Podemos ver que o resultado de 1 - "1" é 0, e isso acontece pois mesmo um dos números 1 está em string o JavaScript reconhece ele como um número por conta do (-) pois não tem como você fazer (string - string).

Terceiro comando

'

Podemos ver que o resultado de 1 + "1" é "11", e isso acontece pois um dos números 1 está em string e então o JavaScript tentando fazer a soma transforma o outro 1 em string também então teoricamente a conta fica = "1" + "1" e isso dá o resultado de "11".
